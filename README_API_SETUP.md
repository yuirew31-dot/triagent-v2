
## ❓ Часто задаваемые вопросы

### В: Какой API выбрать первым?
**О:** Начни с SiliconFlow (полностью FREE), затем добавь Fireworks ($6 баланса хватит на 100+ тестов).

### В: Все ли API работают одновременно?
**О:** Да, проект поддерживает параллельные запросы. Один запрос → нужный API автоматически.

### В: Что если у меня нет денег?
**О:** Используй только FREE API (Gemini 50 req/min, SiliconFlow, Cohere Trial). Это даст 80% функционала.

### В: Как отключить дорогие API?
**О:** Просто не заполняй ключ в .env. Проект автоматически переключится на fallback.

### В: Как отслеживать расходы?
**О:** В каждом API dashboard есть usage stats. Добавь в логи `{ api, tokens, cost }` для мониторинга.

### В: Могу ли я использовать на production?
**О:** Да, но:
1. Ограничь rate limits на бэкенде
2. Кеши результаты (Redis)
3. Мониторь quota per user
4. Используй SiliconFlow как fallback

## 🚨 Частые ошибки

### ❌ Ошибка: "API key missing"
**Причина:** Ключа нет в `.env` или `server/.env` не загружен  
**Решение:** Проверь что файл существует и переменная видна `echo $FIREWORKS_API_KEY`

### ❌ Ошибка: "429 Too Many Requests"
**Причина:** Превышен лимит запросов  
**Решение:** FREE API имеют лимиты. Используй PAID API или жди.

### ❌ Ошибка: "Connection timeout"
**Причина:** API сервер недоступен или интернет выключен  
**Решение:** `curl https://api.fireworks.ai` для проверки доступа

### ❌ Ошибка: "Invalid JSON"
**Причина:** Неправильный синтаксис в запросе  
**Решение:** Используй [API_QUICK_REFERENCE.md](./API_QUICK_REFERENCE.md) для правильного формата
