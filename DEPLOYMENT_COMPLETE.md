## 📱 MOBILE APP

### What is it?
A **Progressive Web App (PWA)** - works exactly like an app you'd download from Play Store, but:
- ✅ No app store needed
- ✅ Instant updates
- ✅ Works offline
- ✅ Smaller file size
- ✅ Always latest version

### How to Install
1. Open the web URL on Chrome (S25+)
2. Tap "Install"
3. Done! App on home screen
4. Works offline

### Features
- ✅ Fullscreen (no browser UI)
- ✅ Offline support
- ✅ Real-time sync online
- ✅ File sharing
- ✅ App drawer integration
- ✅ Push to home screen
