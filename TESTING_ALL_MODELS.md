## 📍 Быстрый старт (5 минут)

### 1. Скопировать .env файл
```bash
cd server
cp .env.example .env
```

### 2. Заполнить API ключи в `server/.env`
```env
FIREWORKS_API_KEY="fw_xxx"
SILICONFLOW_API_KEY="sk_xxx"
GEMINI_API_KEY_1="AIzaSy_xxx"
COHERE_API_KEY="cohere_xxx"
```

### 3. Запустить проект
```bash
npm run dev
```

### 4. Открыть браузер
```
http://localhost:5173
```

## ⚙️ Диагностика проблем

### Проблема: "API key missing"

**Решение:**
```bash
# Проверить, что .env файл загружен
cat server/.env | grep API

# Проверить синтаксис (должны быть кавычки)
FIREWORKS_API_KEY="fw_xxxxx"  ✅ Правильно
FIREWORKS_API_KEY=fw_xxxxx    ❌ Неправильно
```

### Проблема: "429 Too Many Requests"

**Решение:**
```
1. Это нормально для FREE сервисов (Gemini, SiliconFlow)
2. Используй paid альтернативу (Fireworks)
3. Или жди 60 секунд и повтори
```

### Проблема: "Connection timeout"

**Решение:**
```bash
# Проверить доступ в интернет
curl https://api.fireworks.ai

# Проверить точку входа
curl -H "Authorization: Bearer $SILICONFLOW_API_KEY" \
     https://api.siliconflow.cn/v1/chat/completions
```
